## 📖 作品概述
本作品是一款国风2D叙事冒险交互游戏，以千年古桥赵州桥为文化载体，围绕梁思成、林徽因探寻古桥遗踪的故事展开。
将**传统古建筑文化**与**数字交互设计**相结合，包含剧情叙事、平台跳跃、机关解谜、文物收集等玩法，整体风格古韵十足，交互逻辑完整。

## 🎯 设计亮点
- 🎨 **国风视听**：古风音乐+动画转场，打造沉浸式叙事体验
- 🎮 **丰富玩法**：二段跳、移动机关、场景探索、NPC对话交互
- 📱 **云端部署**：导出WebGL版本，GitHub Pages托管，浏览器即开即玩
- 🛠 **性能优化**：视频流式加载、资源预加载、UI自适应，解决WebGL运行卡顿、界面偏移问题

## 🎹 操作按键
| 按键 | 功能 |
| ---- | ---- |
| 方向键 | 角色移动 |
| 空格键 | 跳跃（支持二段跳） |
| E 键/空格键 | 触发剧情/推进对话 |
| Tab 键 | 打开/关闭笔记本面板 |

## 🗺 游戏流程
开幕动画 → 主菜单（音量设置）→ 剧情导语 → 室内剧情场景 → 车马转场动画 → 赵州桥实景探索 → 机关闯关+构件收集 → 古桥修复 → 结局展示

## 💻 技术实现
- 开发引擎：Unity
- 发布平台：WebGL
- 部署服务：GitHub Pages
